# ADP Assignment

## Installation

Clone `git clone https://github.com/sushkum3/ADP-Assignment.git`

Go to directory `cd ADP-Assignment`

Install dependencies: `npm install`

## Usage

Start the project `npm start`

Go to browser or postman and hit http://localhost/
